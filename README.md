# 2D Platform Game

A project featuring a Platform 2D game designed for the subject Game Development

## 🎮 Game controls

AWDS --> Player movement in diferent directions

SPACE --> Player jump and double jump

↑ ↓ → ← --> Shoot fireball in diferent directions

**Debug Keys:**

H --> Show/Hide help menu

F9 --> Debug mode (view collisions)

F10 --> God Mode (in Godmode you are immortal)

F11 --> Enable/Disable FPS limit cap to 30

F1 --> Restart from the first level

F5  --> Save the current game state

F6  -->  Load the previous state

**Note:** F5 saves the game and when its restarted it starts from the point saved, not the initial point. To restart from the beginning press F1 and to load it inside the game, F6.

## 🚀 Features in version 1.0

- **SDL3 Integration**: Latest SDL3 with automatic download and setup
- **Box2D Physics**: Integrated 2D physics engine for realistic movement
- **Tiled Support**: TMX map loading with pugixml
- **Image Loading**: SDL3_image with JPEG and PNG support
- **Cross-platform**: Windows (Visual Studio 2022) and Linux support
- **Automatic Dependency Management**: All libraries downloaded and configured automatically
- **Multi-architecture**: x64 and x86 platform support
- **Performance Tools**: Built-in FPS counter, frame capping, and profiling support
- **Player movement and jumping using Box2D physics**
- **Customized player walk/jump/die animations**
- **Player follow-up camera**
- **Player dies with objects and respawns after die**
- **Player extra movement double jump**
- **1 level implemented**
- **Map designed on tiled with diferent types of colliders and custom tilesets**
- **Game capped at stable 60 FPS**
- **Window title that shows the information: FPS / Avg. FPS / Last-frame MS / Vsync: on/off**
- **Debug options :**
  - **H Shows / hides a help menu showing the keys**
  - **F9 Visualize the colliders / logic**
  - **F10 God Mode (fly around, cannot be killed)**
  - **F11 Enable/Disable FPS cap to 30**
